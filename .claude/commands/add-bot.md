# Add New AI Bot

Follow the instructions in `.claude/skills/add-bot.md` to add a new AI chatbot to the project.

Ask the user for:
1. Bot name (e.g., "ChatGPT", "Claude", "Gemini")
2. Bot URL (e.g., "https://chat.openai.com")
3. New chat shortcut (optional, e.g., "Ctrl+K")

Then follow the skill guide to:
1. Analyze the target page's DOM structure
2. Generate the adapter code
3. Register it in the project
4. Test the integration
